AppVersion Plugin Demo

The AppVersion plugin allows you to read the version of your app from its configuration file and display it to the user.

This may come in handy with f.i. analytics or issue reporting.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android/Windows Phone - App package.

Version

The reported version originates from the config.xml of your app.
